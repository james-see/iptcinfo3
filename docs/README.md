# IPTCInfo3 Documentation

This directory contains the GitHub Pages documentation for IPTCInfo3.

## Viewing Locally

Open `index.html` in your browser:

```bash
open docs/index.html
```

Or use Python's built-in server:

```bash
cd docs
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## GitHub Pages Setup

The documentation is automatically published via GitHub Pages at:
https://jamesacampbell.github.io/iptcinfo3/

## Updating Documentation

1. Edit `docs/index.html`
2. Commit and push changes
3. GitHub Pages will automatically update within a few minutes
